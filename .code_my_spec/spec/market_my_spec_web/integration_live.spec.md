# MarketMySpecWeb.IntegrationLive

OAuth provider connection management for the signed-in user. Lists connected providers (Google, GitHub) with disconnect controls and exposes connect actions that kick off the OAuth flow handled by Integrations. Distinct from UserLive.Settings (which handles email/password/notification settings).

## Type

live_context

## Dependencies

- MarketMySpec.Users
- MarketMySpec.Integrations
