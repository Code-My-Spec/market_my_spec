# MarketMySpecWeb.ThreadLive

LiveViews for browsing ingested engagement threads — list of recently fetched threads per account, drill into a single thread to see normalized OP + comment tree alongside any touchpoint records (drafts or posted comments).

## Type

live_context

## Dependencies

- MarketMySpec.Engagements

## Liveviews

- MarketMySpecWeb.ThreadLive.Index — account-scoped list of recently ingested threads
- MarketMySpecWeb.ThreadLive.Show — single-thread view with OP, comment tree, and touchpoints

## Components

None
