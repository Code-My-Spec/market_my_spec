# MarketMySpec.ProblemDiscovery.Source.Upwork

Upwork Source impl: small Req-based client using app-level UPWORK_API_KEY. Translates a SavedSearch into an Upwork job-search request, paginates, returns JobPosting attribute maps (title, description, total_spent, hire_rate, source_id). First v1 source. See architecture/decisions/problem-discovery-data-sources.md.
