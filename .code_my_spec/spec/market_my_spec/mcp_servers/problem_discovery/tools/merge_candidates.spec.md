# MarketMySpec.McpServers.ProblemDiscovery.Tools.MergeCandidates

MCP tool: merge multiple Candidates into one. Recomputes centroid as mean of combined members, reassigns PaidJobSignal.candidate_id, preserves the winning Candidate's RedTeamVerdict (if any). Agent's refinement surface for over-fragmented clusters.
