# 2026-05-03

The day market_my_spec was born. In a single morning push, John dropped a 314-file initial commit that did not so much start a project as land one fully formed. The scaffold arrived as a Phoenix 1.8 app with magic-link auth, Google and GitHub OAuth, multi-tenant individual-and-agency accounts, invitations, integrations, and a file-backed marketing-strategy skill living under priv/skills. Architecture was boundary-enforced from line one: MarketMySpec for the domain, MarketMySpecWeb for the LiveView and controller layer, and MarketMySpecSpex for behavior. One hundred BDD spex files spread across twelve user stories were committed alongside the implementation, driving through the public surface rather than poking at internals. The harness scaffolding came with it too, decisions and rules and persona research filed under .code_my_spec/, plus a mix spex alias chain that stitches ecto.create, ecto.migrate, compile, and app.start so sexy_spex's bare task could run without a test_helper.

By early afternoon the project pivoted. Story 676 had been built around the agent writing artifacts to its own local filesystem, but that model got rejected in favor of server-side persistence inside the user's MMS account workspace. The marketing-strategy skill content was rewritten across all eight steps to instruct the agent to call write_file with canonical paths instead of reaching for Claude Code's local Write tool, and a new MCP Agent persona was researched and filed on disk to formalize who the skill is talking to. The same commit also swapped sexy_spex from the hex 0.1.0 release to a Code-My-Spec fork on the reusable-givens branch, whose DSL requires `{:ok, context}` returns from every step. That ripple touched roughly a hundred existing spex files, each rewritten to drop the rejected `, context` argument from `scenario` and replace bare `:ok` returns so the suite would compile under the fork.

A sprint day, by any measure. Greenfield in the morning, foundational pivot in the afternoon, and a thousand-line diff across the spex suite to bring everything back into compile.
